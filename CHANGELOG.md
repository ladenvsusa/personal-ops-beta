# 更新记录

## 0.1.0-beta - 2026-07-14

### 新增

- 首页看板，展示运动、睡眠、任务、支出四类核心信息。
- 人际关系模块，支持人物新增、分类筛选、CSV 模板导入。
- 多关系网功能，支持新增命名、切换关系网、拖拽人物节点、添加人物连线。
- 照片上传功能，人物、出行、理财流水支持上传并预览照片。
- 运动模块，支持手动录入身高、体重、心率、步数、热量、睡眠，并显示体重趋势。
- 工作模块，支持任务进度管理和按时间段生成阶段总结 Prompt。
- 理财模块，支持手动流水、CSV 导入、截图待识别占位。
- 出行模块，支持出行记录和出行规划。
- Capacitor Android 工程和 Debug Beta APK 构建脚本。

### 构建

- 应用包名：`com.personal.ops.beta`
- 应用名称：`个人运营Beta`
- Debug APK：`outputs/personal-os-beta-debug.apk`
