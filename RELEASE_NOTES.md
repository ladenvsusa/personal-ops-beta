# 个人运营系统 Beta 0.1.1

这是一次按实际使用反馈整理的体验更新，重点把首页、底部导航、人际、学习、工作、理财和出行模块改得更接近日常可用状态。

## 重点更新

- 首页取消宣传式展示，顶部改为具体时间信息。
- 首页四个看板卡片支持自选指标和排序，可选择步数、睡眠、本周运动时间、运动负荷、支出、体重、进行中任务等项目。
- 底部导航固定在页面底部，各模块内容在固定应用壳内滚动，避免导航条跟着页面跑。
- 人际模块首页改为“最近更新人物、常用关系分类、关系图”三段结构，分类卡片可点击筛选人物。
- 学习模块聚焦想法记录，新增记录反馈和历史记录列表。
- 工作模块增加紧凑任务进度看板，并内置每周二工单提醒、每周四会议材料、每周五周报、每月18日专项奖励制作等周期提醒。
- 理财模块先聚焦手工轻录入和看板，顶部展示日支出、月支出，下方提供快速记账与近7日支出图。
- 出行模块同时支持无边画布式记录和飞书在线文档链接，便于只录基础信息、把详细材料放到线上文档。

## 当前限制

- 高驰、OneDrive、微信、支付宝等自动接入仍是后续目标，目前以本地数据和手工录入为主。
- 当前 APK 仍为 Debug Beta 包，适合自测和需求打磨，不作为正式发布包。
