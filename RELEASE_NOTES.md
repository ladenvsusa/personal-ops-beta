# 个人运营系统 Beta 0.1.0

这是第一个可安装测试版本，用于验证整体信息架构和核心交互。

## 重点功能

- 一打开就是首页看板，快速查看今日状态。
- 底部导航分为：首页、人际、运动、学习、工作、理财、出行。
- 人际关系支持多张关系网，可以新增、命名、切换、拖拽人物节点，并给人物之间添加连线。
- 人物、出行、理财流水支持上传照片并显示缩略图。
- 工作模块可以按时间段生成工作总结 Prompt，方便后续交给 AI 生成阶段总结。

## 当前限制

- 当前数据主要保存在本机本地存储中，卸载应用或清理数据可能导致丢失。
- 高驰、OneDrive、微信/支付宝账单自动导入尚未正式接入。
- 当前 APK 为 Debug Beta 包，适合自测，不适合作为正式发布包。
